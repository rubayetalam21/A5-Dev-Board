Question-1: What are the different ways to select an element in the DOM?

The different ways to select an element in the DOM are
1. getElementById()
2. getElementsByClassName()
3. getElementsByTagName()
4. querySelector()
5. querySelectorAll()

Question-2: What is the difference between innerHTML, innerText, and textContent?

innerHTML: returns the HTML inside of the element
innerText: returns the visible text inside of an HTML element
textContent: returns all the text content inside of an element including hidden text.

Question-3: What is event delegation in the DOM?

Event delegation is a process where a single event listener can be added to a parent element instead of adding multiple listeners to individual child elements.

Question-4: What is event bubbling in the DOM?

Event bubbling is a process where an event starts from the target element and then moves up to its parent elements
